Instalacion de bootstrap 4.3, font Awesome 5.11, uso de giltro con ajax y paginador asp.net
1. Seleccionar la solucion .sln (AppConsultorio.sln) y click derecho, para luego 
seleccionar "Administrar paquetes NuGet"

2. En la busqueda poner "bootstrap 4" donde muestra lo siguiente acerca del paquete 
que se debe instalar:
	Descripcion: The most popular front-end framework for developing responsive, mobile first projects on the web.
	Autores: The Bootstrap Authors, Twitter Inc.
(https://www.youtube.com/watch?v=T3zGyMdK8Uk)

3. En la busqueda poner "font awesome" donde muestra lo siguiente acerca del paquete 
que se debe instalar:
	Nombre: Font.Awesome
	Descripcion: Font Awesome gives you scalable vector icons that can instantly be customized â€” size, 
	color, drop shadow, and anything that can be done with the power of CSS.
	Please see https://fontawesome.com/how-to-use/on-the-web/setup/getting-started?using=web-fonts-with-css for help using the CSS version of Font Awesome.
	Please note that this package is not maintained by the Font Awesome team.
	Autores: Dave Gandy,Tieson Trowbridge (NuGet package only)

4. En la busqueda poner "Ajax, Unobtrusive, jQuery" donde muestra lo siguiente acerca del paquete 
que se debe instalar:
	Nombre: Microsoft.jQuery.Unobtrusive.Ajax
	Descripcion: jQuery plugin that unobtrusively sets up jQuery Ajax.
	Version: 3.2.6
	Autor: Microsoft
	Fecha de publicacion: 05/10/2018
5. En la busqueda poner "PagedList.Mvc" donde muestra lo siguiente acerca del paquete 
que se debe instalar:
	Nombre: PagedList.Mvc
	Descripcion: Asp.Net MVC HtmlHelper method for generating paging control for use with PagedList library.
	Version: 4.5
	Autor: Troy Goode
	Fecha de publicacion: 30/09/2013
	Dependencias: PagedList (>= 1.17.0)
	Por defecto se instala el paquete o dependencia anterior
